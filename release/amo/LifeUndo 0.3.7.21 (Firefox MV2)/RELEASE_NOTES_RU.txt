0.3.7.21 — локализация и UX

• На странице настроек добавлен переключатель языков (EN/RU/HI/ZH/AR), единый тёмный стиль.
• Ссылки из попапа и «Activate VIP» ведут на локализованные страницы сайта.
• Мягкое локальное напоминание через 7 дней о PRO/VIP (без телеметрии).
• Небольшие правки UI; логика 0.3.7.20 без изменений.

Этот релиз сосредоточен на интернационализации и улучшении пользовательского опыта, сохраняя основную функциональность версии 0.3.7.20. Сбор данных и основные функции остаются без изменений.

Ключевые улучшения:
- Поддержка нескольких языков для интерфейса (5 языков)
- Локализованные ссылки на сайт в зависимости от языковых предпочтений
- Мягкое напоминание об обновлении через 7 дней использования (полностью локально)
- Единообразное тёмное оформление всех элементов интерфейса
- Улучшенная доступность и пользовательский опыт

Никаких изменений в основной функциональности, сборе данных или разрешениях.
