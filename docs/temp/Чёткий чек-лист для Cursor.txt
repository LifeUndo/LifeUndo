# Чек-лист для Cursor: вставка `<head>` и зеркала RU/EN

## 0) Общие правила

* Вставлять блоки **сразу после** `<meta charset>` и `<meta name="viewport">`.
* `canonical` указывает на **ту же локаль** (RU → RU, EN → EN).
* `alternate hreflang` всегда тройкой: `ru`, `en`, `x-default` (и указывать зеркальный URL).
* `og:image` везде: `https://lifeundo.ru/assets/lifeundo_og.png`.
* Логотип в разметке JSON-LD `Organization.logo`: `https://lifeundo.ru/assets/lifeundo-round.png`.

---

---

## 10) Техническое

* В корне `docs/` добавить:

  * `robots.txt`:

    ```
    User-agent: *
    Allow: /
    Sitemap: https://lifeundo.ru/sitemap.xml
    ```
  * `sitemap.xml` с основными URL (мы давали шаблон).
  * Пустой файл `docs/.nojekyll` (если ещё нет).
* Проверить, что **Settings → Pages**: Branch `main`, Folder `/docs`.

---

## 11) Проверка (acceptance)

* Для каждой пары RU/EN открыть страницы и проверить:

  * `canonical` указывает на текущую локаль.
  * `alternate` зеркала корректны.
  * `og:image` загружается, 200 OK.
  * Меню/футер: пункт **Фонд/Fund** присутствует, бейджи FK + “We give 10%” отображаются.
* Пройтись в инкогнито: `/`, `/en/`, `/pricing`, `/en/pricing`, `/support`, `/en/support`, `/fund`, `/en/fund`, `/changelog`, `/en/changelog`, `/news`, `/en/news`, `/privacy/ru.html`, `/en/privacy/`, `/success.html`, `/fail.html`.
