## Дополнительно (по желанию)

**BreadcrumbList** (микроразметка хлебных крошек — для внутренних страниц, пример для Pricing RU):

```html
<script type="application/ld+json">
{"@context":"https://schema.org","@type":"BreadcrumbList","itemListElement":[
{"@type":"ListItem","position":1,"name":"Главная","item":"https://lifeundo.ru/"},
{"@type":"ListItem","position":2,"name":"Тарифы","item":"https://lifeundo.ru/pricing/"}
]}
</script>
```

**FAQPage** (на отдельной странице FAQ добавь пару Q\&A блоков):

```html
<script type="application/ld+json">
{"@context":"https://schema.org","@type":"FAQPage","mainEntity":[
{"@type":"Question","name":"Как восстановить текст формы?","acceptedAnswer":{"@type":"Answer","text":"Установите LifeUndo… История форм хранится локально, текст можно вернуть."}},
{"@type":"Question","name":"Чем Pro отличается от Free?","acceptedAnswer":{"@type":"Answer","text":"Free — до 20 элементов… Pro — до 500, быстрый поиск, экспорт/импорт."}}
]}
</script>
```

**robots.txt** (в корне `docs/robots.txt`):

```
User-agent: *
Allow: /
Sitemap: https://lifeundo.ru/sitemap.xml
```
