Политика конфиденциальности

Последнее обновление: 2025-09-08

Кратко

  • LifeUndo хранит данные локально в вашем браузере (chrome.storage.local / browser.storage.local).
  • Нет телеметрии, нет сетевых запросов, нет серверов.
  • Все функции (включая Pro) доступны офлайн.

Какие данные обрабатываются

  • История текста — последние N состояний (20 в Free / 200 в Pro). Хранится только в этом браузере.
  • Закрытые вкладки — адрес, заголовок, favicon для последних N вкладок (10 / 50).
  • Буфер обмена — последние N скопированных текстов (10 / 50).
  • Лицензия — статус и ключ (офлайн-валидация). Ключ не отправляется на внешние сервисы.
  • Локальная статистика — счётчики действий (popup/undo/restore), показываются только вам.

Срок хранения

История и статистика хранятся до тех пор, пока вы не очистите их в настройках расширения или не удалите расширение/данные браузера.

Безопасность

  • Данные не покидают устройство. Мы не агрегируем и не анализируем их на стороне сервера.
  • Экспорт/импорт — по вашему явному действию (JSON-файл).
