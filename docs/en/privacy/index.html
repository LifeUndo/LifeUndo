LifeUndo — Privacy Policy

Last updated: 2025-09-12

TL;DR

  • No data is sent to any servers.
  • Everything is stored locally in your browser (chrome.storage.local / browser.storage.local).
  • No analytics, no telemetry, no ad SDKs, no network requests.

Data we process

  • Text history — last 20 (Free) / 200 (Pro) states.
  • Closed tabs — last 10 (Free) / 50 (Pro) entries (URL, title, favicon).
  • Clipboard history — last 10 (Free) / 50 (Pro) text values.
  • Local stats — counters (popup/undo/restore), stored only on device.

You can clear data in browser settings or uninstall the extension.

Licenses & offline validation

Pro activation happens in the options page and is stored locally. Offline validation uses local storage only. No personal data is transmitted.

Security & access

Only the extension can access its local storage within the browser. We have no remote access to your content.
