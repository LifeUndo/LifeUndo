What is Undo

Undo — "cancel action", go back one step. LifeUndo is your "time machine a few seconds back" for text, tabs and clipboard.

How it works

✍️ Undo text

Stores last 20 (Free) / 200 (Pro) text states.

🧩 Undo tabs

Restores last 10 (Free) / 50 (Pro) closed tabs.

📋 Undo clipboard

Copied text history: 10 / 50.

📊 Local statistics

Action counters — no network, no telemetry.

🔐 Privacy

Data lives in browser.storage.local. Zero network requests.

🔑 Licenses

Offline Pro activation. 7-day trial for new users.
