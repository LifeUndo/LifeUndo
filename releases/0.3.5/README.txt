LifeUndo v0.3.5 - License Import Fix
=====================================

This release focuses on fixing the license import workflow with proper buttons,
live UI updates, and complete RU/EN localization.

Files included:
- LifeUndo-0.3.5-chromium.zip (Chrome/Chromium version)
- LifeUndo-0.3.5-firefox.xpi (Firefox version)
- release_notes_en.md (English release notes)
- release_notes_ru.md (Russian release notes)

Key improvements:
- Fixed license import button workflow
- Live VIP UI updates when license changes
- Complete RU/EN localization for all buttons
- Better error handling and status messages
- Improved integration between Options and Popup

Installation:
1. Chrome: Extract zip and load as unpacked extension
2. Firefox: Install xpi file directly

For detailed information, see release_notes_en.md or release_notes_ru.md
