LifeUndo v0.2.4 — Fixed License Import
=====================================

This release fixes the license import interface and script initialization issues.

## Files

- LifeUndo-0.2.4-firefox.xpi     - Firefox extension (FIXED)
- LifeUndo-0.2.4-chromium.zip   - Chrome/Edge extension

## Installation

Firefox:
1. Download LifeUndo-0.2.4-firefox.xpi
2. Open about:debugging
3. Click "Load Temporary Add-on"
4. Select the downloaded .xpi file

Chrome/Edge:
1. Download LifeUndo-0.2.4-chromium.zip
2. Extract the zip file
3. Open chrome://extensions
4. Enable "Developer mode"
5. Click "Load unpacked" and select the extracted folder

## License Import

1. Open extension settings (about:addons → LifeUndo → Options)
2. Click "Import .lifelic"
3. Select your .lifelic file (JSON or armored format)
4. VIP features activate instantly

## What's Fixed

- Removed duplicate license blocks
- Fixed script initialization
- Cross-browser API support
- Instant UI updates
- Better error handling
