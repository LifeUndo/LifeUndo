GetLifeUndo Apple — Mesh 0.6.1.1
================================

Назначение
----------
Этот файл резервирует слот под будущие GOLD-сборки для экосистемы Apple
(iOS/iPadOS/macOS) в рамках Mesh 0.6.1.1.

Цели и требования
-----------------
- Клиенты Apple должны быть совместимы с моделью лицензии TEAM:
  - до 5 устройств в рамках одной подписки;
  - множество (50+) ключей для пользователей/устройств;
  - единый trial/лицензия с Windows и браузерными расширениями.
- UI и тексты:
  - повторяют основные разделы и смысл Windows/Extensions GOLD 0.6.1.1;
  - ясно объясняют TEAM, лимиты устройств и ключей, B2B/B2G-сценарии.
- Все будущие Apple-клиенты должны использовать общий backend
  (статусы trial/TEAM, список устройств, ключи), как и остальные платформы.

Будущие артефакты
-----------------
- iOS/iPadOS:
  - планируемый пакет: App Store / TestFlight;
  - локальный GOLD-артефакт при наличии:
    - `releases/gold/0.6.1.1/apple/lifeundo-ios-<build>-gold.ipa`.
- macOS:
  - планируемый пакет: notarized .dmg / .pkg;
  - локальный GOLD-артефакт при наличии:
    - `releases/gold/0.6.1.1/apple/lifeundo-macos-<build>-gold.dmg`.

Примечание
----------
До появления реальных сборок эта папка служит только документированным слотом
для Mesh 0.6.1.1, чтобы вся картина GOLD-платформ была консистентной.
