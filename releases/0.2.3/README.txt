LifeUndo v0.2.3 — Minimal License Import
==========================================

This release introduces a simplified license import interface with minimal UI and instant VIP activation.

## Files

- LifeUndo-0.2.3-firefox.xpi     - Firefox extension
- LifeUndo-0.2.3-chromium.zip   - Chrome/Edge extension
- checksums.txt                  - File checksums for verification

## Installation

Firefox:
1. Download LifeUndo-0.2.3-firefox.xpi
2. Open about:addons
3. Click "Install Add-on From File"
4. Select the downloaded .xpi file

Chrome/Edge:
1. Download LifeUndo-0.2.3-chromium.zip
2. Extract the zip file
3. Open chrome://extensions
4. Enable "Developer mode"
5. Click "Load unpacked" and select the extracted folder

## License Import

1. Open extension settings (about:addons → LifeUndo → Options)
2. Click "Import .lifelic"
3. Select your .lifelic file (JSON or armored format)
4. VIP features activate instantly
