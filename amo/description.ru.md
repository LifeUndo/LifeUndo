# GetLifeUndo — верни потерянный текст, вкладки, буфер и скрины

GetLifeUndo сохраняет вашу работу в реальном времени: восстанавливает формы и текст, историю закрытых вкладок, буфер обмена и локальный архив скриншотов. 100% локально — без облака, слежки и телеметрии. Данные остаются только на вашем устройстве.

## Возможности
- Восстановление форм и текста в реальном времени
- История вкладок и возврат случайно закрытых страниц
- История буфера обмена
- Сохранение скриншотов в отдельной папке
- Приватность по умолчанию: никакой отправки данных

## Статус
Активная разработка, регулярные обновления. 7 дней бесплатно.
