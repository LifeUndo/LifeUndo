# 2) Фонд (фикс-политика, что / кому / как)

* **Отчисления**: 10% **чистой выручки** (net revenue).
* **Куда**:

  1. «Privacy for Schools» — мини-гранты на уроки цифровой гигиены, плакаты/методички.
  2. «UX for Privacy» — стипендии студентам/джунам за лучшие UX-кейсы приватности.
  3. «Open-Source Privacy» — донаты в OSS-проекты, повышающие приватность/контроль данных.
* **Как работает**:

  * **Отчёт** — раз в год на `/fund` (с PDF/CSV).
  * **Пул** — ежеквартально распределяем в пропорции 40/30/30 (Schools/UX/OSS).
  * **Заявки** — форма `/fund/apply` (ФИО/команда, страна, краткая суть, бюджет, результат, ссылка/репозиторий). Ответ в 14 дней.
  * **Конфликт интересов** — заявитель не может быть аффилирован с LifeUndo/GetLifeUndo (декларация в форме).
* **Юр. оговорка**: фонд — внутренний грантовый пул компании; это **не** НКО, деньги распределяются по публичным критериям; список получателей публикуется.

# 4) Почему «сейчас ни черта не работает» — краткий разбор

* **lifeundo.ru главная = «баннер оплаты»** — потому что мы сознательно сделали **минимальную VIP-врезку**. Сейчас это выглядит как «баннер». Исправляем: добавляем нормальную шапку/футер/секции, а форму — как блок, а не весь экран.
* **Страница “Поддержите нас” на скрине** — это **старый белый шаблон** (исторический /fund/support). Удаляем/скрываем.
* **Кнопки в расширении 0.3.7.11** — ранее были дубляжи ключей в локалях и ненадёжный код обработчиков; после чистки локалей нужно **пересобрать** и убедиться, что `popup.js` и DOM-идентификаторы совпадают для MV2/MV3. Ещё частая причина: нет элемента `#btnPro` → обработчик не вешается; либо CSP блокирует `window.open`. Чиним патчем ниже и валидируем `web-ext lint` + ручной smoke.

## 5.4 Расширение 0.3.7.11 — починить клики и меню

* В `extension_firefox/popup.html` **гарантировать** наличие элементов с id:

  * `btnVip`, `btnPro`, `lnkWebsite`, `lnkPrivacy`, `lnkSupport`, `lnkLicense`.
* В `popup.js` заменить обработчики на «железобетонные» (MV2/MV3-совместимые, без зависимости от `browser.*`):

```js
(function(){
  const open = (url) => {
    try { chrome.tabs.create({ url }); } catch { window.open(url, '_blank'); }
  };

  const byId = (id) => document.getElementById(id);
  const lang = ((navigator.language||'en').toLowerCase().startsWith('ru')) ? 'ru' : 'en';

  // VIP → /pricing (RU)/en/pricing (EN) или сразу форма VIP на .ru
  const pricing = (lang === 'ru') ? 'https://www.getlifeundo.com/pricing' : 'https://www.getlifeundo.com/en/pricing';
  const vipForm = 'https://lifeundo.ru/#vip'; // якорь блока на главной .ru

  const map = [
    ['btnVip',   () => open(vipForm)],
    ['btnPro',   () => open(pricing)],
    ['lnkWebsite', () => open('https://lifeundo.ru/')],
    ['lnkPrivacy', () => open((lang==='ru')? 'https://lifeundo.ru/privacy' : 'https://www.getlifeundo.com/en/privacy')],
    ['lnkSupport', () => open('https://t.me/LifeUndoSupport')],
    ['lnkLicense', () => open('https://www.getlifeundo.com/docs#license')]
  ];

  map.forEach(([id, fn]) => { const el = byId(id); if (el) el.addEventListener('click', fn, { passive:true }); });
})();
```

* Пройти `npx web-ext lint` и пересобрать XPI.
