# GetLifeUndo — локальный «Ctrl+Z для жизни» (Firefox)

GetLifeUndo сохраняет последние введённые тексты, элементы буфера обмена, быстрые скриншоты по кнопке и недавно закрытые вкладки — **локально, приватно по умолчанию**.

- **Снимки текста**: input/textarea/contenteditable (пароли и file-поля игнорируются).
- **История буфера обмена**: по действию пользователя (copy).
- **Недавно закрытые**: вкладки/окна с безопасными подписями.
- **Быстрый скрин**: по кнопке и хоткею.
- **Без облака по умолчанию**: всё локально; скрины — только по явному действию.

## Установка
- **Firefox (AMO)**: из списка AMO.
- Текущая версия: **0.3.7.32** (MV2, `gecko.id=lifeundo@example.com`).

## Почему приватно
Нет скрытых автоснимков. Нет паролей/`C:\fakepath` в истории.
